# NetEaseMusicMobile
This is a sample web App imitate the NetEaseMusic UX/UI and implemented simple music playback function

# Run
Client
```javascript
  npm install
  npm run serve
```

Server
```javascript
  node app.js
```

# Backend
API support [NeteaseCloudMusicApi](https://docs.neteasecloudmusicapi.binaryify.com/#/)
